# Homebrew Tap for gw-tools

This is the official Homebrew tap for [gw-tools](https://github.com/mthines/gw-tools).

## Installation

```bash
brew install mthines/gw-tools/gw
```

## Usage

After installation, you can use the `gw` command:

```bash
gw --help
```

For more information, visit the [main repository](https://github.com/mthines/gw-tools).
